## Usage

```bash
$ python3 ./bin/giftexchange.py -h
usage: giftexchange.py [-h] -g GROUP -p PARTICIPANTS -r RELATIONSHIPS -x HISTORY [-y YEAR] [-l LOOKBACK] [-c CYCLIC] [-m MATCHES] [-R EXTRA_RELATIONSHIPS] [-s SEED]

options:
  -h, --help            show this help message and exit
  -g GROUP, --group GROUP
  -p PARTICIPANTS, --participants PARTICIPANTS
  -r RELATIONSHIPS, --relationships RELATIONSHIPS
  -x HISTORY, --history HISTORY
  -y YEAR, --year YEAR
  -l LOOKBACK, --lookback LOOKBACK
  -c CYCLIC, --cyclic CYCLIC
  -m MATCHES, --matches MATCHES
  -R EXTRA_RELATIONSHIPS, --extra-relationships EXTRA_RELATIONSHIPS
  -s SEED, --seed SEED
```
